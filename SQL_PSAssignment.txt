1.Write a query which displays the module code and the duration for modules whose duration is less than 6.

Sol. SELECT modulecode, duration FROM modules WHERE duration < 6 ;

2.Write a query which will display the BATCHCODE, TRAINERID, COURSECODE, STARTDATE, CAPACITY 
and BATCHSTRENGTH of the batches handled by the trainer KRISHNAN.

Sol. SELECT b.batchcode, b.trainerid, b.coursecode, b.startdate, b.capacity, b.batchstrength FROM batches AS b , trainers as t WHERE b.trainerid = t.trainerid AND t.trainername LIKE 'KRISHNAN';

--------------------------------

3.Write a query which displays all the module names of the course code 100.

Sol. SELECT m.modulename FROM modules AS m, courses AS c , courseandmodules AS cm WHERE c.coursecode = cm.coursecode AND cm.modulecode = m.modulecode ;

-----------------------------

4.Write a query which displays the module names and course code for the course code 100.

Sol. SELECT m.modulename , c.coursecode FROM modules AS m, courses AS c , courseandmodules AS cm WHERE c.coursecode = cm.coursecode AND cm.modulecode = m.modulecode AND c.coursecode = 100;

----------------------------------

5."Write a query which will display the number of modules present in the course code 100. 
<br/>
Hint: display module name count with alias""NumberofModules"""

Sol. SELECT COUNT(*) AS NumberOfModules FROM modules AS m, courses AS c , courseandmodules AS cm WHERE c.coursecode = cm.coursecode AND cm.modulecode = m.modulecode and c.coursecode = 100;

----------------------------------

6."Write a query which will display all the course code and the number of modules present 
in that particular course.
Hint: display module name count with alias""NumberofModules"""

Sol. SELECT c.coursecode, COUNT(*) AS NumberOfModules FROM modules AS m, courses AS c , courseandmodules AS cm WHERE c.coursecode = cm.coursecode AND cm.modulecode = m.modulecode GROUP BY c.coursecode;

------------------------------------

7.Write a query which will display the course title and fees which has the maximum fees.

Sol. SELECT title, fees FROM courses ORDER BY fees DESC LIMIT 1;

8.Write a query which will display the student id, student name, batch code, place and 
phone number of the students who belongs to Batch '3001'.

Sol. SELECT studentid, studentname, batchcode, place, phone FROM students WHERE  batchcode = 3001;

9.Write a Query which will display the student ID, Student name, Batch code, place and phone number who are from CHENNAI.

Sol. SELECT studentid, studentname, batchcode, place, phone FROM students WHERE place = 'CHENNAI';

10.Write a query which displays the total number of modules which has less than 6 days duration. 
<BR>Hint: display total number of modules with'LESSTHANSIXDAYS' as alias.

Sol. SELECT COUNT(*) AS LESSTHANSIXDAYS FROM modules WHERE duration < 6 ;
